Curve Tool

Build points, preview the curve, export the same JSON shape.

This editor reads the Unity-style curve objects in your level config, lets you move or add points visually, and saves the JSON back with matching keyframe fields. It can display the four line options together in one chart while keeping one active curve selected for editing.

Config

Chart scale is fixed to match the board: X axis 0..100, Y axis 0..1.

File Actions

Point Actions

Click inside the chart to add a point. Drag points to move them. The first and last keyframes keep endpoint slopes of 0. Interior points use the previous and next segment slopes.

Export Rule

inSlope = prev segment
outSlope = next segment

For example, a segment from (0.05, 0.7) to (0.08, 0) produces a slope of (0 - 0.7) / (0.08 - 0.05) = -23.3333333333. That becomes the first point’s outSlope and the next point’s inSlope.
